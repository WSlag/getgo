## PRODUCTION NOTES

### Shooting Tips (Low-Budget / DIY)
- **Phone camera is fine** — iPhone 12+ or Android flagship, 4K if possible
- Use **natural lighting** (near windows) rather than expensive studio lights
- Record in quiet environment, add voiceover in post
- Use Filipino-sounding background music from **YouTube Audio Library** (free)

### Editing Tools (Free/Cheap)
- **CapCut** (free, mobile-first, excellent for Reels)
- **DaVinci Resolve** (free desktop editor)
- **Canva Video** (for motion graphics and text overlays)
- **InShot** (mobile editing for vertical videos)

### Text Overlay Design Rules
- Use **white or orange (#f7941d)** text on dark backgrounds
- Font: Bold/Black weight, minimum 48px for mobile
- Always include **Tagalog subtitles** (many watch without sound)
- Add **GetGo PH logo watermark** in bottom right corner
- End card: Always show logo + karga.ph + @GetGoPH
